# DDs Service - 雷小豆定位服务

一个简单的 GPS 定位页面，用于获取用户位置并分享给雷小豆。

## 部署到 Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/chadlui-code/DDs-Service)

### 手动部署

1. Fork 这个仓库
2. 在 Vercel 导入
3. Deploy！

## 使用

打开部署后的 URL → 允许定位 → 复制位置 → 发给雷小豆

## 本地开发

```bash
npm install
npm start
```
